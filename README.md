
# STM32 Mini Project

This repository contains a mini project developed for the STM32 microcontroller platform. The project includes source code and configuration files necessary for building and running the application.

## Project Structure

- `main.cpp`: Main application source file.
- `stm32_wrapper_gpio.c` and `stm32_wrapper_gpio.h`: GPIO wrapper functions for STM32.
- `Serial.h` and `Serial_Config.h`: Serial communication configuration and functions.
- `PinAsInput.h`, `Pin_Mode_Names.h`: Pin configuration headers.
- `stm32f10x_conf.h`, `stm32f10x_it.h`: STM32F10x configuration and interrupt handlers.
- `stm32map.h`: STM32 memory map definitions.
- `mini-projet-GL3-2024-25.pdf`: Project documentation or assignment details.

## Requirements

- STM32 development board (e.g., STM32F103).
- ARM MDK or compatible development environment.
- STM32CubeMX (optional, for configuration).

## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/AchrefHemissi/Stm32_Mini_Projet.git
   ```


2. Open the project in your preferred STM32 development environment.

3. Build and flash the firmware to your STM32 board.
